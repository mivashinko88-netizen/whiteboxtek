# WhiteBoxTek

Marketing website for White Box Tech — a Microsoft security consulting firm specializing in implementation, optimization, and knowledge transfer across the Microsoft security stack.

## About

WhiteBoxTek provides hands-on technical services for organizations running Microsoft security products. The philosophy: configure it, optimize it, and make sure your team owns it — no ongoing dependency.

### Services

- **Implementation & Configuration** — deploy and configure Microsoft security products
- **Troubleshooting & Optimization** — tune existing deployments for better coverage and fewer false positives
- **Training & Knowledge Transfer** — ensure internal teams can operate independently

### Technologies Covered

- Microsoft Sentinel
- Microsoft Defender suite
- Microsoft Entra ID
- Azure security stack

## Tech Stack

- **Frontend** — HTML, CSS, JavaScript
- **Backend** — Node.js with Express
- **Design** — Inter + JetBrains Mono typography, Bootstrap Icons

## Running Locally

```bash
git clone https://github.com/mivashinko88-netizen/whiteboxtek.git
cd whiteboxtek
npm install
node server.js
```
